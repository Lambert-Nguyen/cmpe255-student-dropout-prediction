HOW TO RUN THE DEMO
===============================================================================

1. Clone or download the repository from the GitHub link above
2. Install dependencies: pip install -r requirements.txt
3. Open the notebooks in Jupyter, Google Colab, or VS Code
4. Run notebooks 01 → 02 → 03 in sequence
5. View generated figures in results/figures/

All data files are included; no additional downloads needed (unless you prefer to pull from UCI directly).
